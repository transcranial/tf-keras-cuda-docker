# tf-keras-cuda-docker

Docker image for Tensorflow and Keras with CUDA support

Requires [nvidia-docker](https://github.com/NVIDIA/nvidia-docker)

- CUDA: 8.0
- cuDNN: 5
- Python: 3.5
- Tensorflow: 0.12.0rc0
- Keras: 1.1.2

Additional libs:
- ipython
- numpy
- scipy
- pandas
- pillow
- scikit-image
- scikit-learn
- pydicom
- h5py
